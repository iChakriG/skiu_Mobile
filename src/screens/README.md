# Screens

| Screen | Purpose |
|--------|--------|
| ProductListScreen | List/search products; header links to Cart, Orders, Settings |
| ProductDetailScreen | Product details, “Add to cart” |
| CartScreen | Cart contents, “Checkout” |
| CheckoutScreen | Shipping address form, place order |
| OrdersScreen | List user orders |
| OrderDetailScreen | Single order details |
| SettingsScreen | Set User ID; shows API base URL |

All use `../navigation/index` for types and navigation.
