# App assets

Add the following for a full build:

- **icon.png** – 1024×1024 app icon
- **splash.png** – Splash screen image (e.g. 1284×2778)
- **adaptive-icon.png** – 1024×1024 Android adaptive icon (foreground layer)

You can use placeholder images to run the app; Expo will use defaults if these are missing in some setups.
